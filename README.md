# Desafio Go Expert - Client Server API

## Requisitos

- Go 1.22+
- SQLite

## Instalação

```bash
go mod tidy
```

## Rodando servidor

```bash
go run server.go
```

## Rodando cliente

Em outro terminal:

```bash
go run client.go
```

## Resultado

O cliente criará o arquivo:

```text
cotacao.txt
```

Formato:

```text
Dólar: 4.9094
```
